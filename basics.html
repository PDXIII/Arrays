Arrays Basic

Bitte erstelle einen Array mit 3 Tiernamen. Gib den gesamten Array in der Konsole aus. Gib den Namen Deines Lieblingstieres in der Konsole aus.

Arrays sind Datenstrukturen die mehrere Werte speichern kann. Dabei kann es sich um homogene, sowie um heterogene Daten handeln.

In Arrays könne wir Zahlen, Strings, Objekte, weitere Arrays, Funktionen und Booleans speichern.

Um an ein Element zu gelangen benutzen wir den Index myArray[i]. Wenn ich eine Funktion innerhalb eines Arrays aufrufen möchte, dann mit Klammern myArray[i]().

Wenn ich eine Element aus einem verschachtelten Array erhalten möchte, dann benötige ich zwei indixes myArray[i][j].
